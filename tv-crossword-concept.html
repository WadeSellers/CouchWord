Navigation Model

  • Swipe left/right/up/down — move between cells
  • Click center — toggle Across / Down
  • Hold Siri button — voice input
  • Play/Pause — clue list overlay
  • Menu — pause / back
  • Shake remote — undo. Single letter: removes that letter. Full word: removes the whole word but preserves pre-existing letters from crossing words. Uses CMMotionManager.
  • Double-tap center — zoom out to full grid overview, tap again to return
Under the hood: Each voice-word entry is logged as a single "action" on an undo stack with a snapshot of which cells were pre-filled. Shake pops the last action and restores only the cells you changed. Letter-by-letter pushes one action per letter.

Puzzle Data & Sources

  • v1: Claude API generates 5x5 crosswords as JSON. Review by hand. Bundle 20 puzzles.
  • v2-3: Bulk generate 100+ with validation script (real words, solvable, no dupes).
  • v4+: Server-side daily puzzle via Cloudflare Worker.
  • v8+: Community submissions via companion iOS app.
  • v12+: On-demand AI generation. User picks theme, size, difficulty.

Tech Stack & Architecture

v1 (Local Only)

SwiftUI (tvOS) + JSON puzzles + UserDefaults + Siri Dictation + CMMotionManager

v4+ (With Server)

SwiftUI (tvOS) → REST API → Cloudflare Worker → D1 Database

v10+ (Multi-platform)

tvOS + iOS companion + CloudKit sync + GameKit + StoreKit 2

Key Dependencies

  • Swift 5.9+ / SwiftUI, tvOS 17+
  • Apple built-in dictation (no Speech framework needed)
  • CoreMotion for shake detection
  • AVFoundation for sound effects
  • Apple Developer account ($99/year)

Accessibility

  • VoiceOver: Every cell, clue, button labeled. Position announced.
  • Screen-off mode (v11): Entire puzzle by voice. Opens crosswords to visually impaired users.
  • High contrast: Bold grid lines, maximum contrast, larger text.
  • Colorblind modes: Protanopia, deuteranopia, tritanopia. Patterns + color for feedback.
  • Font scaling: Dynamic Type support. Fewer cells visible but always readable.
  • Reduced motion: Respects system setting. No animations.
  • Switch control: Full scanning mode for users who can't use the remote.
